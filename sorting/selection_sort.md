How Selection Sort works:

- Start from the first position
- Find the smallest element in the unsorted part
- Swap it with the current position
- Move to the next position
- Repeat until sorted


```c
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i;

        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }

        if (minIndex != i) {
            int temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }
```
